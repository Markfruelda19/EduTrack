# 🎓 EduTrack – Student Record System

EduTrack is a simple web-based Student Record System built using PHP, JavaScript, and MySQL.  
It allows users to manage student information, store grades, and compute average grades automatically.

---

## ✨ Features

- Add, edit, and delete students
- Store and manage student grades
- Search and filter students
- Automatic average grade computation
- Responsive and user-friendly interface

---

## 🛠️ Built With

- PHP
- JavaScript
- MySQL
- HTML & CSS
- Bootstrap

---

## 📂 Database Structure

### Students Table
- Student ID
- First Name
- Last Name
- Course
- Year Level

### Grades Table
- Student ID
- Subject
- Grade

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/edutrack.git
```

2. Move the project folder to your `htdocs` directory

3. Import the SQL database file into phpMyAdmin

4. Start Apache and MySQL using XAMPP

5. Open in browser

```bash
http://localhost/edutrack
```
